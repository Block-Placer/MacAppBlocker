Installation

Run the following commands in the default Terminal app to install Blocker. 

curl -O https://appblock.blockservers.net/AppBlockInstaller.sh
chmod +x AppBlockInstaller.sh
sudo ./AppBlockInstaller.sh

Usage


# To block an app, run block then the path to the application. Some apps are in /System/Applications/
block /Applications/Safari.app

# To view what apps are blocked:
block list

# To unblock an app:
unblock Safari

# To uninstall Blocker:
block uninstall
