Block apps instantly on macOS

Blocker is a lightweight background system that force-closes selected apps the moment they start — even after reboot.

Install Blocker How it works

Why Blocker?

Built for simplicity, reliability, and control — without tracking, servers, or unnecessary complexity.

Always running

Uses a LaunchDaemon to stay active at all times, even before login.

Instant blocking

Blocked apps are terminated immediately when launched.

Terminal simple

Block and unblock apps using clean, easy commands.

How it works

Blocker installs a small background service that watches for blocked applications.

LaunchDaemon

Starts automatically on boot and stays alive.

Block list

App names are stored locally in a protected file.

Process termination

Uses system tools to force-close blocked apps instantly.

Installation

Install Blocker with a single command. Administrator access is required. Run the command below in the default Terminal app. 

curl -O https://appblock.blockservers.net/AppBlockInstaller.sh
chmod +x AppBlockInstaller.sh
sudo ./AppBlockInstaller.sh

Usage

Blocker works entirely from the terminal. 

# Block an app
block /Applications/Safari.app

# List blocked apps
block list

# Unblock an app
unblock Safari

# NOTE: Some apps are in the /System/Applications/ directory! These are usually native MacOS apps.
# Igore syntax errors, just look for if it says your app is blocked or unblocked. Ignore the grep and touch errors.

Safety & transparency

No tracking

Blocker does not log usage or send data anywhere.

Local only

Everything runs locally on your Mac.

Admin removable
